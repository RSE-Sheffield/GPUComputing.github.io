--- layout: post title: GPU and Parallel Computing Using Matlab ---

Matlab is a superb tool for research computing providing a diverse range of toolboxes and development tools. This masterclass workshop provided by Matlab will take place from 10:00 until 13:00 in the Mappin Building, Lecture Theatre 04. This session lead by the developers of the Matlab parallel computing toolbox, will be a masterclass in parallel computing using Matlab. It will cover;

  • Running matlab on GPUs,
  • The parallel computing toolbox and using parfor.
